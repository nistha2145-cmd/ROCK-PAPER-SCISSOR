# 🎮 Rock Paper Scissor Game

An interactive **Rock Paper Scissors** web game built using **HTML, CSS, and JavaScript**. The game allows users to play against the computer with real-time score tracking and an attractive user interface.

## 🌐 Live Demo

🔗 https://nistha2145-cmd.github.io/ROCK-PAPER-SCISSOR/

## 📂 GitHub Repository

🔗 https://github.com/nistha2145-cmd/ROCK-PAPER-SCISSOR

## 🚀 Features

- 🎯 Play Rock, Paper, Scissors against the computer
- 🤖 Random computer choice generation
- 🏆 Real-time score tracking
- 💬 Instant result messages (Win, Lose, Draw)
- 🎨 Responsive and user-friendly interface
- ⚡ Fast and lightweight web application

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
